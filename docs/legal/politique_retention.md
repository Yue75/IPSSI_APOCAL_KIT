# Politique de rétention des données personnelles

**Projet :** EduTutor IA  
**Version :** MVP  
**Conformité :** RGPD (Articles 5, 6, 15, 17 et 20)

---

# 1. Durée de conservation des données

Les données personnelles sont conservées uniquement pendant la durée nécessaire à la finalité pour laquelle elles ont été collectées.

| Type de donnée | Durée de conservation |
|----------------|----------------------|
| Compte utilisateur | Tant que le compte est actif |
| Informations de connexion | Durée du compte |
| Cours importés | Jusqu'à suppression par l'utilisateur ou fermeture du compte |
| Quiz générés | Tant que le compte est actif |
| Réponses aux quiz | Tant que le compte est actif |
| Scores et statistiques | Tant que le compte est actif |
| Signalements | 3 ans |
| Journaux d'audit (logs) | 12 mois |
| Demandes RGPD (SAR) | 3 ans après traitement |

À l'expiration de ces délais, les données sont supprimées ou anonymisées.

---

# 2. Base légale du traitement

Les traitements réalisés par EduTutor IA reposent sur les bases légales prévues par l'article 6 du RGPD.

| Donnée | Base légale |
|---------|-------------|
| Création du compte | Exécution du contrat |
| Génération des quiz | Exécution du contrat |
| Sauvegarde des cours | Exécution du contrat |
| Historique des résultats | Exécution du contrat |
| Logs de sécurité | Intérêt légitime |
| Signalements | Intérêt légitime |
| Gestion des demandes RGPD | Obligation légale |

Les données ne sont jamais revendues à des tiers.

Elles sont utilisées uniquement pour :

- fournir le service ;
- améliorer la qualité des quiz ;
- assurer la sécurité de la plateforme ;
- répondre aux obligations légales.

---

# 3. Suppression des données

Les utilisateurs disposent de leurs droits conformément au RGPD.

Ils peuvent notamment demander :

- l'accès à leurs données (Article 15) ;
- la rectification (Article 16) ;
- l'effacement (Article 17) ;
- la limitation du traitement (Article 18) ;
- la portabilité (Article 20).

En cas de suppression du compte :

- les informations personnelles sont supprimées ;
- les cours importés sont supprimés ;
- les quiz et résultats associés sont supprimés ;
- les données conservées pour obligation légale ou sécurité sont conservées uniquement pendant la durée prévue.

Les demandes de suppression sont enregistrées dans un registre interne (DataRequest) comprenant :

- identité du demandeur ;
- date de la demande ;
- statut de traitement ;
- date de réponse ;
- empreinte (hash) du fichier exporté lorsqu'un export est réalisé.

---

# Conclusion

EduTutor IA applique une politique de conservation fondée sur le principe de minimisation des données du RGPD.

Les durées de conservation sont définies pour chaque catégorie de données, les bases légales sont identifiées et les procédures d'effacement ainsi que les demandes d'accès sont tracées afin d'assurer la conformité du service.
